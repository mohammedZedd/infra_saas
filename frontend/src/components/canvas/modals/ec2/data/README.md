# components/editor/modals/ec2/data

Static catalogue data used by the EC2 configuration wizard.

## Files

| File | Description |
|------|-------------|
| `ec2-amis.ts` | List of curated AMI options (Amazon Linux, Ubuntu, Windows, etc.) with platform/architecture metadata |
| `ec2-instance-types.ts` | List of EC2 instance types with vCPU, memory, storage, and pricing data |
