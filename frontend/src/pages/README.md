# pages

Top-level route page components. Each file maps to a route defined in `App.tsx`.

## Files

| File | Route | Description |
|------|-------|-------------|
| `Landing.tsx` | `/` | Public marketing landing page |
| `Register.tsx` | `/register` | Account registration page |
| `Dashboard.tsx` | `/dashboard` | Project management dashboard |
| `Editor.tsx` | `/editor/:projectId` | Visual infrastructure canvas editor |
| `ProjectDetail.tsx` | `/project/:projectId` | Project detail view with tabs |
